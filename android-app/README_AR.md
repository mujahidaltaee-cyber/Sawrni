# صورني — Android APK Test App

هذا المشروع يحوّل آخر نسخة من موقع صورني إلى تطبيق Android حقيقي للتجربة.

## ما الذي يفعله التطبيق؟

- يفتح آخر نسخة من الموقع: `https://sawrni-demo.atwebpages.com/?v=4`
- يحافظ على تسجيل الدخول والكوكيز.
- يدعم تسجيل العميل ومقدم الخدمة.
- يدعم الحجوزات، الإدارة، الإعلانات، رفع الصور والملفات.
- يدعم زر الرجوع داخل صفحات الموقع.
- يدعم تنزيل الملفات.
- يعرض شاشة Offline عند انقطاع الإنترنت.
- يستخدم هوية صورني V4 في الأيقونة وشاشة البداية.

## فتح الإدارة من داخل التطبيق

اضغط 7 مرات بسرعة خلال 5 ثوانٍ على الزاوية العلوية اليسرى داخل التطبيق. ستفتح بوابة الإدارة المخفية.

يمكن أيضاً فتح الرابط المخصص:

`sawwarni://management`

## تغيير رابط الموقع

عدّل الملف:

`app/src/main/java/com/sawwarni/app/AppConfig.java`

ثم غيّر `BASE_URL` و`MANAGEMENT_URL`.

## إنشاء APK بواسطة GitHub Actions

1. ارفع محتويات هذا المشروع إلى جذر GitHub Repository.
2. افتح تبويب **Actions**.
3. افتح **Build Sawwarni APK**.
4. اضغط **Run workflow**.
5. بعد اكتمال البناء افتح التشغيل الناجح.
6. من قسم **Artifacts** نزّل `Sawwarni-Test-APK`.
7. داخل ZIP ستجد `Sawwarni-Test-v1.apk`.

## تثبيت APK على Android

1. انقل APK إلى هاتف Android.
2. افتحه.
3. فعّل السماح بالتثبيت من هذا المصدر إذا طلب Android ذلك.
4. ثبّت التطبيق وافتحه.

هذه نسخة Debug للاختبار وليست مفتاح نشر نهائي على Google Play.
