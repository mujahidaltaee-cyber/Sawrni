# Sawwarni Android Test App

A lightweight native Android WebView wrapper for the live Sawwarni PHP/MySQL platform.

- Package: `com.sawwarni.app.debug`
- Website: `https://sawrni-demo.atwebpages.com/?v=4`
- Minimum Android: 7.0 (API 24)
- Target SDK: Android 15 (API 35)
- Build output: `Sawwarni-Test-v1.apk`

The GitHub Actions workflow builds the APK automatically. See `README_AR.md` for exact steps.
